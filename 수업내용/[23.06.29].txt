- HighLevelInterface 아래 Input 클래스 생성

	enum class eKeyCode
	{
		Q, ~ M, END
	}
	// 키의 상태
	enum class eKeyState
	{
		Down,
		Up,
		Pressed,
		None
	};
	class Input
	{
	public :
		struct Key
		{
			eKeyCode code;   // 
			eKeyState state; // 키의 상태
			bool bPressed;   // 누름 여부
		}
		static void Initialize()
		static void Update()
		
		// static 으로 만들어야 객체를 만들지 않고도 사용할 수 있다.
		// 캡슐화
		static bool GetKeyDown(eKeyCode code)
		{
			return mKeys[(int)code].state == eKeyState::Down;
		}
		static bool GetKeyUp(eKeyCode code)
		{
			return mKeys[(int)code].state == eKeyState::Up;
		}
		static bool GetKey(eKeyCode code)
		{
			return mKeys[(int)code].state == eKeyState::Pressed;
		}
	
	private :
		static std::vector<Key> mKeys;  // 키에 대한 정보드릉 저장
	}
		

[ enum ]
- 숫자를 문자로 치환할 수 있는 문법
- enum class 하고 enm 은 동일하다 다만 이름이 겹칠까봐 class 로 사용해준다.

[ static ]
- 정적변수
- 전역변수와 같다 다만 클래스 내부에서만 동작하는 전역변수인 것
- 접근제한 지정자로 외부에서의 접근 여부를 설정할 수 있다.
- 데이터 영역에 들어간다 
- 클래스 안에서만 동작하는 전역변수를 만들고자 할때 사용하는 키워드
- static 변수는 외부에서 초기화를 해줘야 한다!
